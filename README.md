**Smart Resume Analyser** is an industrial-grade, AI-driven career intelligence suite. Unlike basic ATS checkers, SRA deconstructs professional DNA using Large Language Models to provide a strategic roadmap for career acceleration.

---

## 🚀 Key Features

- 🧠 **ATS Neural Audit:** Mathematical and semantic analysis of resumes against Job Descriptions (JDs) with sub-second inference.
- 🛰️ **Strategic Roadmap Architect:** Generates a 10-year professional trajectory including salary growth and milestone actions.
- 🔗 **LinkedIn Neural Audit:** Standalone social branding optimization tool to upscale professional digital presence.
- 💬 **AI Architect Chatbot:** A 24/7 real-time career coach powered by Llama 3.1.
- 🎭 **Dual-Persona Logic:** Custom intelligence reports based on whether you are a **Recruiter** or a **Job Seeker**.
- 🌌 **Cinematic UX:** High-end dark theme with starfield animations, glassmorphism, and smooth layout transitions.

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Database:** SQLAlchemy (SQLite)
- **Intelligence Engine:** Groq Cloud API (Llama 3.3-70B & 3.1-8B)
- **NLP & Parsing:** SpaCy, pdfplumber, PyPDF2
- **Frontend:** Tailwind CSS, Jakarta Sans, Outfit Typography
- **Animation:** HTML5 Canvas, CSS3 Keyframes
